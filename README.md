# JDE

JDE is an intelligent systems studio. This repository contains its one-page
contact and capabilities site: a focused introduction to the AI operations,
data, performance, and automation systems JDE builds for modern businesses.

## Local development

```bash
npm install
npm run dev
```

## Validation

```bash
npm test
```

## Deployment

Pushes to `main` are built and deployed automatically to GitHub Pages through
the workflow in `.github/workflows/deploy-pages.yml`.

Live site: <https://enerjadez.github.io/JDEportfoliogpt/>
